# EDA-top-1000-IMBD
Exploratory Data Analysis – IMDb Top 1000 Movies
Este projeto realiza uma Análise Exploratória de Dados (EDA) sobre o dataset IMDb Top 1000 Movies, contendo informações dos filmes mais bem avaliados da plataforma.
O foco é entender padrões, distribuições e relações entre variáveis relevantes, apoiando tomadas de decisão baseadas em dados.
Objetivo:
Explorar características que definem filmes bem avaliados, analisando notas, duração, receita, popularidade, gêneros e tendências ao longo do tempo.
Dataset:
O dataset inclui informações como:
Nota do IMDb
Metascore
Ano de lançamento
Duração do filme
Receita (Gross)
Número de votos
Gêneros, diretores e elenco
Etapas da Análise:
Carregamento e inspeção inicial do dataset
Análise de dimensões, tipos de dados e valores nulos
Limpeza e padronização de colunas numéricas
Tratamento de dados ausentes de forma criteriosa
Estatísticas descritivas antes e após a limpeza
Análises Realizadas
Distribuição das notas do IMDb
Relação entre nota, metascore e receita
Análise da duração dos filmes
Evolução dos lançamentos ao longo dos anos
Frequência de gêneros, diretores e atores
Impacto da popularidade (número de votos) nas avaliações
Ferramentas Utilizadas:
Python
Pandas
NumPy
Matplotlib
Seaborn
Estrutura:
O projeto está organizado em um notebook Jupyter, com seções bem definidas para cada etapa da análise, facilitando leitura e reprodução.
Observações:
Este projeto tem foco exclusivamente exploratório.
